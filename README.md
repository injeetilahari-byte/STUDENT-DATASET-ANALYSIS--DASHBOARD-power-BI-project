 Student Dataset Analysis Dashboard

* Built an interactive Power BI dashboard to analyze and visualize student academic data.

* Helps in understanding and comparing academic performance across different branches.

 Key Features

* Displays the total number of students.
  
* average attendance percentage branch-wise.
  
* Tracks pending fee amounts for each department.
* 
* Highlights average CGPA* across branches.
  
* Includes *interactive slicers/filters for branch-wise analysis.
  
* Uses KPI cards, bar charts, and pie charts  for better visualization.

 Key Insights

* easier to compare student performance between branches.
  
* Helps identify attendance patterns and academic trends.
 
* Provides quick visibility into fee-related records.


 🎨 Dashboard Design

* Created with a clean and structured layout.

* visually balanced theme for a professional dashboard appearance.

 ✅ Outcome
